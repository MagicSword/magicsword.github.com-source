Static site using hugo by golang
---

# Hugo 指令

```bash
$hugo version   #hugo版本

$hugo new site quickstart

#config.toml 是配置文件檔

$hugo server -D

$hugo new content/posts/my-first-post.md
```

hugo  重新建立新站 ，記得新檔取消 draft

在 `public` 下 ，輸入 

```bash
$ git push -u origin master
```

```bash
$ git init

```
